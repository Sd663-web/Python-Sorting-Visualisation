# Python Sorting Visualizer

**Python Sorting Visualizer** is an interactive application that demonstrates **Bubble, Insertion, Merge, and Quick Sort** in real-time. Users can **adjust speed, pause/resume**, and **compare execution times**, helping visualize algorithm behavior step by step.

---

## Features

- Visualizes **Bubble, Insertion, Merge, and Quick Sort**  
- **Real-time execution** with highlighted comparisons and swaps  
- **Interactive controls**: speed adjustment, pause/resume, generate new arrays  
- Tracks **progress and elapsed time** for each algorithm  
- Ideal for **learning and teaching sorting algorithms**
